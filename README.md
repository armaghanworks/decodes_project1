# Muhammad Armaghan Khan — Portfolio

A personal portfolio website for Muhammad Armaghan Khan, a Software Engineering student, showcasing his education, skills, projects, and extracurricular activities.

## About

This is a single-page, responsive portfolio site built with plain HTML and CSS (no frameworks or build tools). It highlights academic background, technical skills, featured projects, leadership activities, and contact information.

## Sections

- **Hero** — Introduction and quick links to Projects and Contact
- **About** — Education history and academic background
- **Skills** — Technical skills, tools, design/media, and languages
- **Projects** — Featured projects with descriptions, tags, and links to their repositories
- **Activities** — Leadership and extracurricular involvement
- **Contact** — Email, GitHub, LinkedIn, and location

## Running Locally

No build steps or dependencies are required.

1. Clone the repository:
   ```bash
   git clone https://github.com/armaghanworks/decodes_project1.git
   cd decodes_project1
   ```
2. Open `index.html` directly in your browser, or serve it locally:
   ```bash
   npx serve .
   ```

## Deployment

This is a static site and can be deployed with **GitHub Pages**:

1. Go to the repository's **Settings → Pages**
2. Under **Source**, select the `main` branch and `/ (root)` folder
3. Save — the site will be published at `https://armaghanworks.github.io/decodes_project1/`
